# Tampermonkey Universal AI Prompt Commands BG

Българска версия на Tampermonkey скрипт за по-бърза работа с чатове с изкуствен интелект.

Скриптът заменя универсалните тригери `Q1–Q10` с готови AI промптове. Тези тригери не са обвързани с език: потребителят може да замени `Q1`, `Q2` и останалите със свои думи, команди или фрази.

## За какво служи

Скриптът помага за бързо въвеждане на готови промптове в ChatGPT, Gemini, Claude, Copilot и други AI чатове. Вместо всеки път да се пише дълга инструкция, се въвежда `Q1` и скриптът поставя пълния промпт.

## Как работи

Скриптът следи активното текстово поле. Ако цялото съдържание на полето точно съвпада с един от тригерите `Q1–Q10`, то се заменя с подготвения промпт.

```text
Q1
```

се заменя с промпт за превод на български език.

```text
Q8
```

се заменя с промпт за извличане на важни факти.

Обикновен текст не се променя. Например `Q1 друг текст` няма да бъде заменен.

## Собствени тригери

Тригерите могат да се променят в кода, в обекта `COMMANDS`.

```javascript
'Q1': `...`
```

може да стане:

```javascript
'PREVEDI': `...`
```

`Q1–Q10` са само стандартните универсални тригери.

## Къде може да се използва

- ChatGPT
- Google Gemini
- Claude
- Microsoft Copilot
- други сайтове с текстово поле

Скриптът съдържа:

```javascript
// @match        *://*/*
```

## Изискване преди инсталиране

Първо в браузъра трябва да бъде инсталирано разширението **Tampermonkey**. Скриптът се инсталира в Tampermonkey, а не в GitHub и не в конкретен сайт. GitHub служи само за съхранение на файла `.user.js`.

## Бърза инсталация

1. Инсталирайте Tampermonkey.
2. Отворете Raw линка:

```text
https://raw.githubusercontent.com/1777maxim7771/bg_tampermonkey-universal-ai-prompt-commands/main/tampermonkey-universal-ai-prompt-commands.user.js
```

3. Потвърдете инсталацията в Tampermonkey.
4. Отворете AI чат и въведете `Q1`.

## Инсталация чрез GitHub

Отворете файла `tampermonkey-universal-ai-prompt-commands.user.js`, натиснете **Raw** и потвърдете инсталацията в Tampermonkey.

## Импорт чрез URL

В Tampermonkey отворете **Dashboard → Utilities → Import from URL**, поставете Raw линка и потвърдете.

## Ръчна инсталация

Създайте нов скрипт в Tampermonkey, поставете кода от `.user.js` файла и запазете.

## Защо Tampermonkey разпознава скрипта

Tampermonkey разпознава заглавката `// ==UserScript==` и разширението `.user.js`.

## Команди по подразбиране

- `Q1` — превод на български език.
- `Q2` — обобщение на текст.
- `Q3` — резюме на писмо в един ред.
- `Q4` — превод на прост немски A2-B1.
- `Q5` — поправка на български текст.
- `Q6` — кратък официален отговор.
- `Q7` — просто обяснение на текст.
- `Q8` — извличане на важни факти.
- `Q9` — списък с необходими действия.
- `Q10` — официално писмо на немски език.

## Проверка

Въведете `Q1` в AI чат. Ако скриптът работи, `Q1` ще бъде заменен с пълния промпт.

## Възможни проблеми

Проверете дали скриптът е включен, страницата е презаредена, `Q1` е въведено без допълнителен текст, Tampermonkey има разрешение за сайта и курсорът е в редактируемо поле.

## Цел на проекта

Да ускори повтарящата се работа с AI чатове: превод, обобщение, анализ на писма, официални отговори и обработка на документи.
