# Tampermonkey Universal AI Prompt Commands BG

Tampermonkey скрипт за замяна на кратки команди с готови AI промптове в AI чатове.

Скриптът заменя само точни команди с готови промптове. Обикновеният текст не се променя. ChatGPT, Gemini, Claude, Copilot.

## Команди

- `BG1` — Преведи изцяло и точно предоставения текст на български език.
- `BG2` — Обобщи предоставения текст на български според смисъла и контекста.
- `BG3` — Направи на български кратко тематично резюме на писмото строго в един ред.
- `BG4` — Преведи предоставения текст на прост и ясен немски език, ниво A2-B1.
- `BG5` — Поправи предоставения текст на български.
- `BG6` — Напиши кратък, учтив и официален отговор на това писмо на прост немски език, ниво A2-B1.
- `BG7` — Обясни с прости думи на български какво означава текстът.
- `BG8` — Извлечи всички важни факти и ги структурирай на български: хора, организации, адреси, дати, срокове, суми, номера на документи, изисквания, решения, задължения, документи и следващи стъпки.
- `BG9` — Състави на български ясен списък с необходимите действия според текста.
- `BG10` — Състави въз основа на текста учтиво официално писмо на прост немски език, ниво A2-B1.

## Инсталиране

Инсталирайте Tampermonkey, отворете връзката към userscript файла, потвърдете инсталирането и въведете първата команда в AI чат.

[Userscript](https://raw.githubusercontent.com/1777maxim7771/bg_tampermonkey-universal-ai-prompt-commands/main/tampermonkey-universal-ai-prompt-commands.user.js)

Скриптът се инсталира в разширението Tampermonkey, а не в GitHub или сайт. GitHub само съхранява файла.

Tampermonkey → Dashboard → Utilities → Import from URL:

`https://raw.githubusercontent.com/1777maxim7771/bg_tampermonkey-universal-ai-prompt-commands/main/tampermonkey-universal-ai-prompt-commands.user.js`
